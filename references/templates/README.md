## 关键格式规范（跨文件统一）

- **仿宋体**：投委会Word材料（二号/三号/小四号）
- **宋体**：投资建议书、风控回复、预审回复（二号标题/四号正文/小四号表格）
- **表格**：白底黑框，线条宽度一致，跨页固定表头
- **行间距**：投委会材料30磅，其他24磅
- **PE预测**：20-40倍整数区间，超范围需说明
- **财务单位**：万元，不保留小数点
- **16:9宽屏**：PPT专用格式
