# 附件0：立项建议书（PPT）

**阶段**：立项上会
**版本**：2026年3月版（宽屏16:9）
**优先级评级**：A/B/C

## 10 章结构

| 章节 | 内容 | 关键要求 |
|------|------|---------|
| 1. 公司概况 | 基本信息、组织架构、股权结构、融资历史、团队分析 | 实控人出资情况、关联交易详述；团队5维度评价 |
| 2. 商业模式 | 产品/服务收入拆解表（三年一期）、主要客户、主要供应商、关键客户份额分析 | 刚需性、议价能力、差异化、价格及毛利率趋势 |
| 3. 行业与市场 | SWOT、产业链、发展周期、政策、市场容量（产品级TAM）、竞争分析表、产业协同、产业落地 | 细分产品市场而非全产业链；TOP5竞品必覆盖 |
| 4. 核心技术 | 技术来源、技术领先性及佐证、研发投入、研发团队薪酬vs行业 | 含科量匹配度分析 |
| 5. 财务数据及盈利预测 | 资产负债（三年一期）、经营成果（三年一期）、盈利预测（覆盖退出期）、营收构成预测、毛利率预测 | 异常变动需解释；毛利率/净利率偏低/现金流异常重点解释 |
| 6. 投资方案及资金用途 | 融资金额、增资/老股比例、估值谈判空间、业绩承诺、回购保障、资金使用计划 | — |
| 7. 投资回报预测 | PE 20-30-40倍退出回报矩阵、上市可行性分析（板块/标准符合性/题材属性） | PE超40倍需说明 |
| 8. 投资亮点 | 赛道层面 + 标的层面（差异化/稀缺性）+ 估值层面 | 高度凝练 |
| 9. 投资风险 | 风险大表：风险描述/风险程度/风险影响/风险应对 | 含风险评估和应对分析 |
| 10. 投资策略符合性分析 | 九环标准（预期回报/企业家/商业模式）+ 八维投资审美 | 逐条"是否符合+理由" |

## 2026版主要调整

- 实控人及团队分析力度提升，关注团队出资
- 增加产品运用场景、刚需性、议价能力维度
- 市场容量、竞争对手分析维度固化
- 技术来源、领先性佐证、研发薪酬含科量匹配度
- 盈利预测周期拉长至退出时点
- 放开PE倍数限制（>40倍需说明）
- 新增风险大表（评估+应对）
- 新增九环标准+八维投资审美
- 宽屏16:9版式

## 实际写作风格（参考34个历史样板）

- **投资亮点前置**：在第2-4页先展示，再进入目录和正文
- 目录简化为4-5章
- 每页页脚"达善天下 晨见未来"
- 每页一个核心论点，用具体数据支撑
- 句式："XXXXX，印证了/体现了/说明了 YYYYY"
- 符号体系：Ø（主要观点）、•（支撑细节）、➢（子层级）

## 格式

- 首页标题：微软雅黑32号加粗
- 正文标题：仿宋体32号
- 表格：白底黑框，仿宋体12号
