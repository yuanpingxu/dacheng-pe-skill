# 附件1：投委会决策议案（Word）

**阶段**：投决上会
**定位**：根据投资建议书归纳提炼，≤15页，精炼语言
**附件依赖**：投资建议书 + 主投经理声明

## 6 章结构

| 章节 | 内容要点 | 写作提示 |
|------|---------|---------|
| 一、项目背景介绍 | 核心业务概要 | 2分钟以内，简洁明了，详细信息放第三章 |
| 二、项目所处行业概况 | ①行业演变/现状/趋势 ②市场规模及成长性（驱动+限制因素）③竞争格局 ④行业痛点+布局逻辑 ⑤投资时点判断 ⑥产业协同效应 ⑦产业落地 | 给投委会建立行业认知 |
| 三、拟投企业概况 | ①工商信息/团队/融资历史 ②商业模式论证（解决痛点/成熟度/可持续性/客户供应商）③投资核心逻辑（行业地位/技术先进性/竞对优劣势/差异化）④近三年财务指标 ⑤未来规划+盈利预测 ⑥投资亮点 ⑦投资风险 | 风险需针对性、实质性，不要普适宽泛 |
| 四、投资逻辑总结 | 行业+项目要点归纳提炼 | 亮点+风险基础上，结合行业做归纳 |
| 五、投资方案 | ①投前估值/融资总额/投后估值 ②我方金额/股比 ③联合投资方 ④业绩保障/回购 ⑤董事监事/一票否决/转让限制 ⑥投资收益测算 | 核心条款 |
| 六、投资安排 | 完成投资时间建议 | 具体基金由投委会决定 |

## 格式规范

- 标题：仿宋体二号加粗
- 正文大标题：仿宋体三号加粗
- 正文/小标题：仿宋体三号
- 表格：白底黑框，仿宋体小四号，跨页固定表头
- PE预测：20-40倍整数标准区间
- 行间距：固定值30磅

## 对应PPT模板

投委会PPT模板为16:9宽屏，6章结构与Word版保持一致：
1. 项目背景介绍
2. 项目所处行业概况
3. 拟投企业概况
4. 投资逻辑总结
5. 投资方案
6. 投资安排

格式：首页标题微软雅黑32号加粗，正文仿宋体32号，表格仿宋体12号。
