# 辅助模板文件

## 附件3：风控报告反馈意见（Word）

**用途**：项目组逐条回复风控部门意见

### 结构
1. **汇总表**：意见类型 / 风险点描述 / 是否认同（是/否/部分）/ 不认同原因
   - 综合风控报告 — 重大风险
   - 综合风控报告 — 一般风险
   - 财务风险
   - 法律风险
2. **一、综合风控报告风险点具体回复**（逐条：风险N → 项目组回复）
3. **二、财务风控报告风险点具体回复**（逐条 + 盈利预测情况）
4. **三、法律风控报告风险点具体回复**（逐条）

### 格式：宋体二号标题/三号正文标题/四号正文/小四号表格，行间距24磅

### 回复风格要点
- **认同时**：简洁直接 + 补充理解和边际判断
- **部分认同时**：先说明认同的部分，再解释不认同的理由，标注"客观事实+主观判断"
- **不认同时**：用数据反驳，附支撑材料
- 每种回复都是微型论证文（问题→事实→分析→结论）

## 附件5：预审会反馈意见（Word）

**用途**：项目组逐条回复预审会意见

### 结构
- 逐条：预审问题一/二/三... → 项目组回复
- 结尾：项目组成员（主投经理第一位）、回复日期

### 与风控回复的区别

| 维度 | 风控回复 | 预审回复 |
|------|---------|---------|
| 问题来源 | 风控部门报告中的风险点 | 预审会委员的提问 |
| 问题粒度 | 按风险类型分类（综合/财务/法律） | 直接按提问编号 |
| 汇总表 | 有（前置） | 无 |
| 回复长度 | 中等（500-1000字/条） | 较长（1000-3000字/条） |
| 侧重点 | 风险缓释 | 业务逻辑验证+确定性分析 |
| 表格使用 | 较少 | 较多（产能表、订单表、竞品表） |

### 格式：与附件3相同（宋体二号标题/三号标题/四号正文/小四号表格，24磅）

## 附件6：主投经理声明（Word）

**用途**：主投经理诚信承诺书

### 4条承诺
1. 恪守诚实守信，认真负责
2. 全面审慎调查，对材料真实性和完整性负全部责任
3. 不存在私下交易或违反八项禁令的事项
4. 已披露一切关联事项（如有）

## 投委会议案-PPT模板（PPT）

**用途**：投委会上会演示，内容与附件1（投委会决策议案）保持一致

### 6章结构（16:9宽屏）
1. 项目背景介绍
2. 项目所处行业概况
3. 拟投企业概况
4. 投资逻辑总结
5. 投资方案
6. 投资安排

### 格式
- 首页标题：微软雅黑32号加粗
- 正文标题：仿宋体32号
- 表格：白底黑框，仿宋体12号
- 内容必须与投委会决策议案Word版一致
