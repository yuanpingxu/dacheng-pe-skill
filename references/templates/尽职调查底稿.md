# 附件4：尽职调查底稿（Word）

**阶段**：尽调阶段
**版本**：2026年3月版
**定位**：尽调操作手册+记录模板，全流程覆盖

## 五大组成部分

### 一、尽调访谈要求（前置规范）

#### 形式层面要求（分三阶段递进）

| 访谈对象 | 立项阶段 | 风控进场前 | 风控核查阶段 |
|---------|---------|-----------|------------|
| 核心高管 | 企业核心高管 | 企业核心高管 | **全部**企业高管 |
| 股东 | 前三大股东 | 10%以上全部股东 | 10%以上全部股东 |
| 部门负责人 | 核心业务负责人 | **全部**部门负责人 | **全部**部门负责人 |
| 场地走访 | 主要场地 | 主要场地走访完毕 | 主要场地走访完毕 |
| 客户 | 2家(TOP3) | 3家(TOP5) | **至少5家**（线下≥4） |
| 供应商 | 2家最重要 | 3家最重要 | **至少4家**（线下≥3） |
| 竞争对手 | 1家(TOP3) | 2家(TOP5) | **至少3家** |
| 行业专家 | 1名 | 2名 | **至少2名** |
| 中介机构 | 1家(如有) | 会计师/律师/券商 | 会计师/律师/券商 |
| 达晨合伙人 | 行业线主管合伙人 | 行业线主管合伙人 | **至少1位**达晨合伙人 |
| 已投企业访谈 | 达晨已投相关企业 | 达晨已投相关企业 | 达晨已投相关企业 |

#### 实质层面要求（2026年新增）
重点围绕五大主题：**成长性、含科量、稀缺性、安全性、真实性**

#### 留档要求
原则上均需录音留档（涉军涉密除外），客观记录，忌讳选择性记录

### 二、访谈前应获取的10项资料
1. 商业计划书
2. 行业状况/竞争态势/发展趋势
3. 历史沿革及股权结构图
4. 公司组织架构图
5. 最近三年一期财务报表
6. 各类业务/产品收入结构表
7. 各期主要客户销售和应收余额
8. 各期主要供应商采购和应付余额
9. 各期销售合同/采购合同台账
10. 盈利预测

### 三、企业内部访谈（8个角色）

| 序号 | 访谈对象 | 核心主题数 | 关键维度 |
|------|---------|----------|---------|
| (一) | 公司整体情况介绍 | 9项 | 业务定位、职责划分、历史沿革、产品矩阵、内部交易、规范性、中介机构、IPO进展 |
| (二) | 董事长/实控人 | 10项 | 个人背景/诚信、业务发展历程、战略定位、客户/销售策略、行业趋势、竞争格局、业绩预算、IPO计划、**投"人"角度总体判断** |
| (三) | 生产负责人 | 5项 | 技术工艺先进性、质量管理、产能产量、合规性(环保/安全)、**业绩真实性核查** |
| (四) | 采购负责人 | 6项 | 供应商依赖、核心物料受限、供应商稳定性、采购单价趋势、资金占用、**采购-销量匹配度核查** |
| (五) | 销售负责人 | 6项 | 销售策略/推广难度、重要客户、行业环境/竞争、合同/收入/客户趋势、**业务数据辅助业绩核查**、盈利预测复核 |
| (六) | 技术负责人 | 5项 | 技术来源及独立性、先进性及差异化(对标竞对)、持续研发能力、新技术进展、**含科量水平判断** |
| (七) | HR负责人 | 5项 | 核心人员稳定性、人才培养/招聘难度、股权激励、绩效考核变化、**业绩真实性核查** |
| (八) | 财务负责人 | 5项 | **业绩真实性**、真实及持续盈利能力、财务/内控规范性、财务风险、盈利预测+资金需求复核 |

每个角色模板含：访谈记录表（可多次）+ 访谈目标/主题 + 访谈记录提示 + 访谈小结

### 五、合伙人面谈记录 + 文件汇总

- 合伙人面谈（3位，各可多次访谈）
- 目标公司提供的其他重要文件及信息汇总

## 关键原则

- 每次访谈必须写小结：验证之前信息一致性 + 新发现重点问题
- 所有模板包含"选择理由"和"访谈资源来源及独立性评价"
- 客观记录，禁止选择性记录正面信息

## 访谈记录写作风格

- **客观纪实**，不美化，保留原始对话风格
- 每项数据标注来源和验证状态
- 资料获取标注"是/否/不适用"，未获取需说明原因
- 小结三个提示必答：①信息一致性验证 ②是否实现访谈目标 ③新发现重要问题
