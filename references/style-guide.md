# 达晨财智投资材料写作风格指南

**来源**：34+个历史项目样板文件（湃肽生物、凯思凯迪、纽瑞特、百奥几何、赜灵、核欣迅明、桥济生物、衡昱生物、锦江电子、苏州中科、维达康、麦肯伟、甘草医生、博纳精密、微纳芯等）

---

## 一、通用写作原则

### 1.1 语气基调
- **自信但不浮夸**：有判断力，每个判断都有数据支撑
- **客观平衡**：亮点和风险都充分展示
- **直白务实**：不堆砌专业术语，让投委会快速理解
- **产业视角**：从产业链角度分析，不局限于单一公司

### 1.2 数字使用习惯
- 金额：亿元/万元，保留1-2位小数
- 增长率：百分比，保留1位小数
- 员工数/专利数：精确到个位
- 日期：YYYY年MM月格式
- 财务正式文档：万元为单位，不保留小数点

### 1.3 核心高频表述

| 场景 | 常用表述 |
|------|---------|
| 市场地位 | "国内领先"、"全球/国内...领军企业"、"第一梯队" |
| 技术评价 | "自主可控"、"国内首家/首款"、"填补...空白"、"国际先进水平" |
| 团队评价 | "深耕行业XX年"、"具备丰富的产业经验"、"学术背景深厚" |
| 增长描述 | "高速增长"、"稳定持续增长"、"复合增长率XX%" |
| 风险描述 | 直接、具体、不回避，包含"内卷"、"洗牌"等直白词语 |
| 投资判断 | "极大概率"、"确定性高"、"值得关注" |

### 1.4 数据呈现偏好
- 表格 > 纯文字（财务数据、竞品对比、管线进展必用表）
- 时间序列至少三年一期
- 中外市场数据并列对比
- 退出回报用PE矩阵

## 二、立项建议书写作风格

### 2.1 与模板的差异

| 模板要求 | 实际做法 |
|---------|---------|
| 10章顺序 | **投资亮点前置**（第2-4页），再进入目录和正文 |
| 目录 | 简化为4-5章：行业→业务→财务→投资方案 |
| 格式 | 每页页脚"达善天下 晨见未来" |

### 2.2 投资亮点写作
- 每页一个核心论点，用具体数据支撑
- 句式："XXXXX，印证了/体现了/说明了 YYYYY"
- 例："美国巨头礼来市值突破万亿美元，印证了GLP-1赛道的广阔成长空间"
- 符号体系：Ø（主要观点）、•（支撑细节）、➢（子层级）

### 2.3 表格使用
- 竞品对比表：成立时间/创始人/团队/市场份额/价格/业绩/融资/估值/优势/劣势
- 融资历史表：轮次/时间/投资人/金额/投后估值
- 管线表：管线/药物/具体进展
- 退出回报表：PE 20-30-40×矩阵

## 三、投资建议书写作风格

### 3.1 摘要（Executive Summary）写法
- **1-2页密集段落**，不分点，一气呵成
- **定位句开篇**：先定义赛道再定位公司
- **四要素必备**：赛道定义+政策/趋势背书 → 核心能力概述 → 关键财务数据 → 投资建议
- 页脚：每页标注"机密 第X页 YYYY-MM-DD"

### 3.2 各章写作特征

**公司概况**：基本信息表格+企业简介段落(200-300字) → 股权结构表(含历史融资) → 组织架构表(部门/负责人/人数/职能) → 管理团队(每人独立段落)

**行业分析**：政策驱动优先(中央到地方逐级列举) → 中美对比叙事 → 产业链三层框架(底层→平台→应用层)

**竞争优势**：按产品线逐一分析 → 全球首创/国内独家等差异化表述 → 竞对对比表

**投资回报**：PE 20-30-40倍矩阵 → 退出时点对应IPO时间线 → 考虑后续稀释

### 3.3 高频句式
- "XX是...的技术革命"、"XX是...的新质生产力代表"
- "降维打击"、"全球首创"、"快速占领30%以上全球市场份额"
- "超预期跨越式成长"、"持续超预期成长"
- "中国XX行业成功的连续创业者"、"以自有资金持续投入超亿元"

## 四、尽职调查底稿写作风格

### 4.1 公司概况写法
- 第一段：一句话定位（成立时间+赛道+核心能力+关键资质）
- 例："浙江湃肽生物股份有限公司成立于2015年...是国内多肽规模化生产的领军企业之一"
- 必须包含：国家级资质、专利数量、员工规模、学历结构

### 4.2 访谈记录风格
- **客观纪实**，不美化，保留原始对话风格
- 每项数据标注来源和验证状态
- 资料获取标注"是/否/不适用"，未获取需说明原因

### 4.3 访谈小结风格
三个提示必答：
1. 对访谈前了解到的信息进行验证，判断是否一致
2. 是否实现访谈目标
3. 新发现的重要问题，后续尽调重点关注

### 4.4 业务数据呈现
- 三年一期数据（收入、毛利率、增长率）
- 区分产品线分别描述
- 例："2022-2025年1-10月，实现营业收入1.48亿、1.75亿、1.84亿和1.17亿元，每年保持10%-20%稳定增长"

## 五、行业研究写作风格

### 5.1 结构特点
- 开篇引用权威观点
- 技术原理→发展历程→市场规模→应用场景→竞争格局
- 大量引用来源标注："Source: McKinsey Global Institute"

### 5.2 语言风格
- 学术+产业并重，既讲科学原理也讲商业价值
- 中英文混用："使能技术（enabling technology）"
- 四阶段发展历程是常见框架
- 善于对比："与传统微生物发酵相比...本质是对细胞干预的定向性"

## 六、风控报告原文风格（风控部门视角）

### 6.1 报告类型对比

| 维度 | 综合风控 | 财务风控 | 法律风控 |
|------|---------|---------|---------|
| 核心问题 | 该不该投？ | 业绩真假？持续性？ | 合规否？有雷否？ |
| 篇幅 | 15-25页 | 40-50页 | 15-20页 |
| 表格密度 | 中等 | 极高（50+表） | 低 |
| 评级输出 | 推荐/不推荐 + 五星 | 重大/一般风险 | ★★★★星级 |
| 典型语气 | 判断性 | 分析性 | 规范性 |

### 6.2 综合风控关键特征
- **评级体系**：推荐 / 有条件推荐 / 不推荐
- **五星评估**：成长性 ★★★、稀缺性 ★★★、含科量 ★★★、安全性 ★★、回报预估 ★
- **语言风格**：冷静怀疑、"不认可，产业化不确定性高"、"穿透包装"

### 6.3 财务风控关键特征
- 数据表格驱动，每页至少1-2个数据表
- 产品级分析：每个产品独立追踪研发投入/收入/毛利/销量趋势
- 生命周期视角：研发→量产→增长→成熟→衰退

### 6.4 法律风控关键特征
- 事实-分析-方案三段式：基本事实→问题分析（引用法规）→风险解决方案
- 重要程度星级标注
- 历史跟踪：前轮投资协议履行情况逐条核查

## 七、风控回复写作风格（项目组视角）

### 7.1 回复态度分类

| 态度 | 写法 |
|------|------|
| **认同** | 简洁直接 + 补充理解和边际判断 |
| **部分认同** | 先说明认同部分 → 再解释不认同理由 → 标注"客观事实+主观判断" |
| **不认同** | 用数据反驳，附支撑材料 |

### 7.2 常见风控关注与回复模式

| 风控关注 | 回复模式 |
|---------|---------|
| 业绩成长性 | ①历史业绩验证 ②新产品管线 ③选品逻辑阐述 ④市场规模+市占率推演 |
| 盈利能力 | ①毛利率趋势分析(分产品、分阶段) ②降本逻辑 ③新产品替代逻辑 |
| IPO合规 | ①中介机构阵容 ②工作进展 ③整改措施+时间节点 ④监管沟通 |
| 关联交易 | ①历史背景说明 ②当前处理方案 ③约束条款 |
| 市场空间 | ①选品策略解释 ②新产品拓展 ③平台能力 |

### 7.3 写作特点
- **数据密度极高**：每个回复都有具体数字支撑
- **产品级分析**：按产品逐一展开
- **前瞻性判断**：不仅回复当下，还描绘未来3年情景
- **就事论事**：尊重风控，用事实说话

## 八、预审回复写作风格

### 8.1 与风控回复的区别

| 维度 | 风控回复 | 预审回复 |
|------|---------|---------|
| 问题来源 | 风控部门报告风险点 | 预审会委员提问 |
| 汇总表 | 有（前置） | 无 |
| 回复长度 | 中等（500-1000字/条） | 较长（1000-3000字/条） |
| 侧重点 | 风险缓释 | 业务逻辑验证+确定性分析 |

### 8.2 典型回复模式
1. **产能数据**：列表展示已投产车间/产线/产能
2. **产品管线图**：已量产→中试→在研 三色标注
3. **市场空间+竞争格局表**：逐个产品列市场规模+竞对进度
4. **意向订单**：客户名/数量/金额明细表
5. **成长模型总结**：从单品驱动→矩阵式成长的逻辑

## 九、立项PPT vs 投资PPT 关键区别

| 维度 | 立项建议书PPT | 投资建议书PPT |
|------|-------------|-------------|
| 投资摘要页 | 无 | **有**（1页，5-7条Bullet） |
| 投后复盘 | 无 | **有**（二轮专属，实际vs预测对比） |
| 九环标准+八维审美 | **有** | 无（已内化） |
| 章节数 | 10章 | 5章（大幅合并） |
| 可比公司估值 | 无 | **有**（国际+国内对标） |
| 叙事逻辑 | 赛道大→公司小→值得研究 | 公司能力→行业验证→回报→风险可控 |
| 财务深度 | 三年一期大表 | 精选关键数字+投后对比 |

## 十、BP（商业计划书）的"脱水"技巧

BP是创业者视角的推销材料，阅读时需要穿透包装：

| BP表述 | 应核查 |
|--------|--------|
| "全球领先" | 核查市场份额数据 |
| "正在研究" | 核查实际研发进度和投入 |
| "全产业链" | 核查各环节是否真有实质业务 |
| "与XX合作" | 核查合作深度（战略协议 vs 实际收入） |
| "国内首家/唯一" | 专利检索+竞品技术对比 |
| "一站式解决方案" | 是否每个环节都有落地案例 |

## 十一、外部专业报告的使用方式

| 报告类型 | 利用方式 |
|---------|---------|
| 外部律所LDD | 作为内部法律风控的交叉验证 |
| 外部财务DD（四大） | 识别会计调整项，对比管理层报表差异 |
| 共同局限 | 依赖公司提供资料，不独立验证 |
