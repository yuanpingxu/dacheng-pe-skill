# 投后管理

## 五维监控体系

| 维度 | 核心跟踪指标 |
|------|------------|
| 经营 | 收入 vs 预算 / 毛利率 / 现金流 / Runway / 行业 KPI |
| 合规 | 董事会召开 / 重大事项报批 / 财报按时出具 / 关联交易审批 |
| 团队 | 核心人员变动 / 员工流动率 / 招聘进度 / 创始人关系 |
| 市场 | 市场份额 / 新进入者 / 客户关系 / 政策变化 |
| 融资 | 下轮进度 / 估值趋势 / 可比估值 / 退出窗口 |

## 三色预警机制

| 维度 | 🟢 绿灯 | 🟡 黄灯 | 🔴 红灯 |
|------|---------|---------|---------|
| 收入 | ≥预算 90% | 预算 70-90% | <预算 70% |
| 现金流 | Runway >18月 | 6-18月 | <6月 |
| 毛利率 | 稳定或改善 | 下降<5pp | 下降>5pp |
| 客户集中度 | Top1<30% | 30-50% | >50% |
| 核心团队 | 稳定 | 1人离职 | 多人离职/创始人矛盾 |
| 合规 | 无异常 | 小问题 | 重大诉讼/处罚 |
| 下轮融资 | 按计划推进 | 延迟3-6月 | >6月无进展 |

### 触发动作
- 🟡 单黄灯 → 加强关注
- 🟡 多黄灯 → 书面解释 + 改善计划
- 🔴 单红灯 → 紧急董事会
- 🔴 多红灯 → 危机预案

## 投后输出物

### 季报模板
企业名称 / 投资时点 / 持股 / 估值 → 本期指标 vs 预期 vs 上期 → 重大事项 → 风险预警 → 下步建议

### 退出分析（至少每半年一次）
退出时机 × 退出路径（IPO/并购/转让）× 预期回报 vs 投资时预期 × 退出障碍清单

## 注意事项

- 务实前瞻、预警导向
- 既要发现价值也要发现问题
- 配合 private-equity/portfolio-monitoring 和 private-equity/returns-analysis 使用
