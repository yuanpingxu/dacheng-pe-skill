# 投后管理

## 五维监控体系

| 维度 | 核心跟踪指标 |
|------|------------|
| 经营 | 收入 vs 预算 / 毛利率 / 现金流 / Runway / 行业 KPI |
| 合规 | 董事会召开 / 重大事项报批 / 财报按时出具 / 关联交易审批 |
| 团队 | 核心人员变动 / 员工流动率 / 招聘进度 / 创始人关系 |
| 市场 | 市场份额 / 新进入者 / 客户关系 / 政策变化 |
| 融资 | 下轮进度 / 估值趋势 / 可比估值 / 退出窗口 |

## 三色预警机制

| 维度 | 🟢 绿灯 | 🟡 黄灯 | 🔴 红灯 |
|------|---------|---------|---------|
| 收入 | ≥预算 90% | 预算 70-90% | <预算 70% |
| 现金流 | Runway >18月 | 6-18月 | <6月 |
| 毛利率 | 稳定或改善 | 下降<5pp | 下降>5pp |
| 客户集中度 | Top1<30% | 30-50% | >50% |
| 核心团队 | 稳定 | 1人离职 | 多人离职/创始人矛盾 |
| 合规 | 无异常 | 小问题 | 重大诉讼/处罚 |
| 下轮融资 | 按计划推进 | 延迟3-6月 | >6月无进展 |

### 触发动作
- 🟡 单黄灯 → 加强关注
- 🟡 多黄灯 → 书面解释 + 改善计划
- 🔴 单红灯 → 紧急董事会
- 🔴 多红灯 → 危机预案

## 投后输出物

### 季报模板
企业名称 / 投资时点 / 持股 / 估值 → 本期指标 vs 预期 vs 上期 → 重大事项 → 风险预警 → 下步建议

### 退出分析（至少每半年一次）
退出时机 × 退出路径（IPO/并购/转让）× 预期回报 vs 投资时预期 × 退出障碍清单

## 注意事项

- 务实前瞻、预警导向
- 既要发现价值也要发现问题
- 配合 private-equity/portfolio-monitoring 和 private-equity/returns-analysis 使用

## 模板与格式

投后管理目前无标准化Word/PPT模板（达晨9个模板中不含投后管理专门模板），但应遵循以下格式规范：

### 季报输出格式
- 企业名称 / 投资时点 / 持股比例 / 当前估值
- 本期关键指标 vs 预期 vs 上期（表格形式）
- 重大事项（公司治理/业务进展/融资/合规）
- 风险预警（三色标注）
- 下步建议

### 退出分析报告格式
- 退出时机判断（当前窗口/1年后/2-3年后）
- 退出路径可行性（IPO/并购/转让 × 概率）
- 预期回报 vs 投资时预期（IRR/MOIC 对比）
- 退出障碍清单 + 清除方案
- 建议：立即退出 / 观望等待 / 继续持有

### 格式规范
- 宋体四号正文
- 白底黑框表格
- 三色预警使用 🟢🟡🔴 标注
- 财务数据精确到万元

## 写作风格

参见 [references/style-guide.md](../references/style-guide.md)：
- **务实前瞻**：预警导向，提前发现信号而非事后解释
- **客观对比**：实际 vs 预测 vs 上期，三维对比
- **数据驱动**：每个判断有数字支撑
- **不回避问题**：红旗信号必须标注，给出处理建议

## 与其他角色的衔接

投后管理发现的重大问题可能需要触发：
- → 角色2（项目尽调）：补充尽调
- → 角色3（投资决策）：后续轮次投资/退出决策
- → 角色4（风控回复）：风控关注事项跟踪回复
