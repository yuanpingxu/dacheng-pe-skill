# 项目分析与尽调

## 阶段一：BP 评估 / 项目初审

### 10 点快速扫描框架
1. 一句话概要（做什么，解决什么问题）
2. 团队分析（创始人背景、团队完整性、行业经验）
3. 产品/技术（成熟度、壁垒、差异化）
4. 商业模式（如何赚钱，UE 是否成立，LTV/CAC）
5. 市场分析（规模、增速、渗透率）
6. 竞争分析（主要竞品对比，护城河在哪）
7. 财务分析（收入/利润/现金流趋势，burn rate）
8. 融资分析（估值合理性、资金用途、历史投资者质量）
9. 亮点 vs 风险（正反并列）
10. 下一步建议 + Deal breaker

### 输出与结论
- `[项目名称] 投资评估报告 — [日期]`
- 🟢 推进尽调 / 🟡 待补充信息 / 🔴 放弃

## BP/PPT 交叉验证评估（数据核查模式）

触发条件：用户发送 PPT/BP 并要求"评估""核对""验证""找问题"时自动进入。

### 六步工作流
1. **提取声称数据**：逐页提取市场规模/增长率/排名/财务/客户/技术声称
2. **外部数据交叉验证**：Web搜索+可比对标+公开信息+融资数据+专利检索
3. **标注可信度**：✅可信 / ⚠️存疑 / 🔴不实 / ❓无法验证
4. **逻辑一致性检查**：规模×份额=收入？增速vs行业？毛利率vs可比？
5. **问题汇总与严重程度**：🔴致命 / 🟡重要 / 🟢轻微
6. **输出 Excel 评估报告**：Sheet1逐页评估明细 + Sheet2综合评分汇总

### 评估维度评分表
| 维度 | 满分 |
|------|------|
| 市场数据可信度 | 20 |
| 竞争格局可信度 | 20 |
| 财务数据可信度 | 20 |
| 技术声称可信度 | 15 |
| 团队背景可信度 | 10 |
| 估值合理性 | 15 |
| **总分** | **100** |

## 阶段二：系统尽调

### 四大模块
| 模块 | 核心问题 | 主要手段 |
|------|---------|---------|
| 业务 CDD | 收入真实、客户质量、市场地位 | 客户/竞品/专家访谈、后台数据核查 |
| 财务 FDD | 收入合规、费用完整、关联交易 | 穿行测试、流水核查、函证、税务核查 |
| 法务 LDD | 股权清晰、知产归属、合同风险 | 工商调档、专利检索、合同审阅 |
| 技术 TDD | 技术先进性、代码质量、团队实力 | 技术访谈、专利评估、代码审查 |

### 尽调底稿格式
核查事项 / 核查方式 / 获取资料 / 核查结论 / 异常情况 / 风险等级

### 尽调报告结构
执行摘要 → 分模块展开 → 风险汇总矩阵 → 附录

## 注意事项

- 初审阶段：冷静质疑，不因 BP 好看就放松
- 尽调阶段：事实导向、怀疑精神、不回避问题
- 财报检查：自动调用 minesweeper 技能扫描异常信号
- 配合 private-equity/dd-checklist 和 financial-analysis/audit-xls 使用

## 模板与格式

### 达晨尽调模板体系
项目尽调的核心模板是 **[附件4：尽职调查底稿](../references/templates/尽职调查底稿.md)**（Word，2026年3月版），五大组成部分：

1. **尽调访谈要求**（前置规范）：三阶段递进（立项→风控进场前→风控核查），实质层面围绕成长性/含科量/稀缺性/安全性/真实性
2. **访谈前应获取的10项资料**：BP、行业报告、财务报表等
3. **企业内部访谈**（8个角色模板）：实控人/生产/采购/销售/技术/HR/财务负责人，每个含访谈目标+记录提示+小结
4. **企业外部访谈**：客户5家/供应商4家/竞对3家/中介机构/专家/已投企业
5. **合伙人面谈记录**：3位合伙人，各可多次

### BP 评估输出
- Excel 文件：`[YYYY-MM-DD]-BP评估-[项目名称].xlsx`
- Sheet 1：逐页逐项评估明细（页码/声称内容/声称类型/PPT数据/外部验证结果/数据来源/偏差/可信度/严重程度/问题描述）
- Sheet 2：综合评估汇总（市场/竞争/财务/技术/团队/估值 6维度打分，满分100）
- 结论：🟢可推进 / 🟡需补充尽调 / 🔴建议放弃

### 尽调底稿格式
- 核查事项 / 核查方式 / 获取资料 / 核查结论 / 异常情况 / 风险等级
- 访谈记录：客观纪实风格，标注信息来源和验证状态
- 资料获取标注"是/否/不适用"，未获取需说明原因

### 写作风格
参见 [references/style-guide.md](../references/style-guide.md) 第四节"尽职调查底稿写作风格"：
- 公司概况：一句话定位（成立时间+赛道+核心能力+资质）
- 访谈记录：客观纪实，不美化，保留原始对话风格
- 访谈小结必答三问：信息一致性验证 / 访谈目标是否实现 / 新发现重要问题
- 业务数据：三年一期，分产品线描述，具体数字
