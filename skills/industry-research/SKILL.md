# 行业研究与赛道梳理

## 模式判断

根据任务复杂度自动选择：
- **深度模式**：完整行业研究报告（定义→规模→产业链→格局→驱动→启示）
- **快速模式**：赛道扫描/投资地图（细分→评分→Mapping→时机→推荐）

## 深度模式工作流

### Step 1: 行业定义与边界
- 界定上下游，明确行业边界
- 避免边界过宽或过窄
- 一句话定义行业

### Step 2: 市场规模与增长驱动
- TAM / SAM / SOM 测算
- 历史 3 年 CAGR，未来 3-5 年预测
- 增长因子拆解（需求/政策/技术）

### Step 3: 产业链与利润池
- 上游→中游→下游各环节梳理
- 各环节价值分配比例
- 利润池迁移趋势判断

### Step 4: 竞争格局
- CR5 / CR10 集中度
- 头部玩家对比表（收入/份额/增速/差异化）
- 竞争关键要素与进入壁垒评估

### Step 5: 驱动因素
- 需求端 / 政策端 / 技术端
- PEST 落地到具体影响

### Step 6: 投资启示
- 赛道是否值得投（是/否/观望）
- 关键观察点（3-5 个）
- 推荐关注标的（3-5 家 + 理由）

## 快速模式工作流

### Step 1: 赛道定义与细分方向
- 按技术路线 / 商业模式 / 应用场景切分

### Step 2: 赛道吸引力评分矩阵
- 市场空间 × 增速 × 竞争烈度 × 进入时机 × 退出通道
- 加权打分

### Step 3: 标的 Mapping 大表
- 公司名 / 细分方向 / 融资轮次 / 估值 / 核心指标 / 亮点 / 风险

### Step 4: 投资时机判断
- 概念期 / 成长期 / 成熟期 / 整合期

### Step 5: 推荐关注标的
- 优先跟踪 3-5 家及理由

## 输出格式

- 深度模式：`[行业名称] 行业深度研究报告 — [日期]`
- 快速模式：`[赛道名称] 投资地图 — [日期]`
- 语气：专业客观、数据驱动

## 注意事项

- 数据标注来源，不确定处明确标注
- 深度模式接近券商行研质量；快速模式高效直接
- 配合 equity-research 和 china-market-researcher skills 使用

## 模板与格式

### 深度模式输出结构
参见 [references/templates/README.md](../references/templates/README.md) 了解达晨模板体系。行业研究报告虽无固定模板，但需遵循以下格式：
- 标题：宋体小二号加粗居中
- 一级标题：宋体四号加粗
- 正文：宋体四号
- 表格：白底黑框，统一宽度
- 行间距：固定值 24 磅
- 所有数据标注来源

### 写作风格
参见 [references/style-guide.md](../references/style-guide.md) 第五节"行业研究写作风格"：
- 开篇引用权威观点建立 credibility
- 技术原理→发展历程→市场规模→应用场景→竞争格局
- 学术+产业并重，中英文混用（如"使能技术 enabling technology"）
- 四阶段发展历程是常见框架
- 大量引用来源标注："Source: McKinsey Global Institute"
