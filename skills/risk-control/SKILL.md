# 风控报告回复

## 回复五步法

1. **理解确认**：用自己的话重述风控关注点
2. **事实陈述**：相关数据、尽调发现、合同条款
3. **分析论证**：为什么风险可控（行业对标/历史参照/逻辑推演/概率评估）
4. **缓释方案**：交易条款层面 + 投后管理层面 + 交割条件层面，附时间表和责任人
5. **残余风险判断**：缓释后剩余敞口是否在可接受范围

## 常见风控问题速查

| 问题类型 | 回复核心思路 |
|---------|------------|
| 收入真实性 | 穿行测试 + 客户访谈 + 银行流水 |
| 客户集中度 | 行业特征论证 + 客户粘性 + 分散化计划 |
| 估值合理性 | 多方法交叉 + 可比对标 + 增长假设论证 |
| 关联交易 | 定价公允性 + 占比趋势 + 清理计划 |
| 知识产权 | FTO 分析 + 归属协议 + 历史无纠纷 |
| 团队稳定性 | 竞业+锁定+关键人保险+接班人计划 |
| 退出路径 | IPO/并购可行性 + 可比退出案例 + 回报敏感性 |
| 政策风险 | 政策趋势 + 合规程度 + 压力测试 |
| 对赌可行性 | 历史业绩 vs 目标 + 行业参照 + 补偿能力 |

## 输出格式

- 逐条对应：`风控问题 #N → 回复要点 → 支撑材料索引`
- 末尾附"未解决风险清单"（诚实标注无法完全消除的风险）

## 注意事项

- 尊重理性、就事论事，不争辩，用数据和事实说话
- 对合理关切要承认并回应
- 参考样板风格：逐条回复结构、论证模式
