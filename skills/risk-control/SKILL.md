# 风控报告回复

## 回复五步法

1. **理解确认**：用自己的话重述风控关注点
2. **事实陈述**：相关数据、尽调发现、合同条款
3. **分析论证**：为什么风险可控（行业对标/历史参照/逻辑推演/概率评估）
4. **缓释方案**：交易条款层面 + 投后管理层面 + 交割条件层面，附时间表和责任人
5. **残余风险判断**：缓释后剩余敞口是否在可接受范围

## 常见风控问题速查

| 问题类型 | 回复核心思路 |
|---------|------------|
| 收入真实性 | 穿行测试 + 客户访谈 + 银行流水 |
| 客户集中度 | 行业特征论证 + 客户粘性 + 分散化计划 |
| 估值合理性 | 多方法交叉 + 可比对标 + 增长假设论证 |
| 关联交易 | 定价公允性 + 占比趋势 + 清理计划 |
| 知识产权 | FTO 分析 + 归属协议 + 历史无纠纷 |
| 团队稳定性 | 竞业+锁定+关键人保险+接班人计划 |
| 退出路径 | IPO/并购可行性 + 可比退出案例 + 回报敏感性 |
| 政策风险 | 政策趋势 + 合规程度 + 压力测试 |
| 对赌可行性 | 历史业绩 vs 目标 + 行业参照 + 补偿能力 |

## 输出格式

- 逐条对应：`风控问题 #N → 回复要点 → 支撑材料索引`
- 末尾附"未解决风险清单"（诚实标注无法完全消除的风险）

## 注意事项

- 尊重理性、就事论事，不争辩，用数据和事实说话
- 对合理关切要承认并回应
- 参考样板风格：逐条回复结构、论证模式

## 达晨模板体系

风控阶段使用三个辅助模板，参见 [references/templates/辅助模板.md](../references/templates/辅助模板.md)：

### 附件3：风控报告反馈意见（Word）
- **结构**：汇总表（意见类型/风险点/是否认同/不认同原因）→ 综合风控回复 → 财务风控回复 → 法律风控回复
- **格式**：宋体二号标题/三号正文标题/四号正文/小四号表格，行间距24磅

### 附件5：预审会反馈意见（Word）
- **结构**：逐条预审问题→项目组回复，无汇总表
- **格式**：同附件3
- **与风控回复的区别**：问题来源不同（委员提问 vs 风控报告）、回复更长（1000-3000字 vs 500-1000字）、侧重点不同（业务逻辑验证 vs 风险缓释）

### 附件6：主投经理声明（Word）
- **4条承诺**：诚实守信 / 材料真实性 / 无私下交易 / 已披露关联事项

## 回复写作风格

参见 [references/style-guide.md](../references/style-guide.md) 第七节"风控回复写作风格"和第八节"预审回复写作风格"。

### 三种态度回复方法
- **认同**：简洁直接 + 补充理解和边际判断
- **部分认同**：先说明认同部分 → 再解释不认同理由 → 标注"客观事实+主观判断"
- **不认同**：用数据反驳，附支撑材料

### 论证特点
- 数据密度极高：每个回复都有具体数字支撑
- 产品级分析：按产品逐一展开
- 前瞻性判断：不仅回复当下问题，还描绘未来3年情景
- 就事论事：尊重风控，用事实说话
- 完整叙事：每个回复是微型论证文（问题→事实→分析→结论）

### 预审回复特有模式
预审会常问"确定性"问题，回复五件套：
1. 产能数据（已投产车间/产线/产能列表）
2. 产品管线图（已量产→中试→在研 三色标注）
3. 市场空间+竞争格局表（逐个产品）
4. 意向订单明细（客户/数量/金额）
5. 成长模型总结（从单品驱动→矩阵式成长）
